koa2实现数据爬虫

2018-03-06

Koa2制作爬虫的全过程，包括项目建立，使用superagent获取源数据、使用cheerio解析，Json数据返回。

项目建立

安装脚手架

npm install -g koa-generator

创建项目

koa2 koaCrawler

运行项目

cd koaCrawler
npm install
npm start
浏览器访问http://localhost:3000/

superagent应用

安装

npm install superagent

使用

cheerio应用
